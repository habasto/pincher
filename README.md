# Pincher

**Pincher** es una herramienta open source para Linux que permite monitorear procesos usando `pidstat` y guardar métricas en **SQLite**.
Ideal para crear historiales de CPU y memoria de procesos específicos, con enfoque en análisis y contribuciones de la comunidad.

## Características

- Monitoreo de uno o varios procesos.
- Captura métricas de CPU, memoria y uso de recursos.
- Guarda automáticamente en SQLite con timestamp.
- CLI fácil de usar.
- Extensible para exportación CSV/JSON y visualización futura.
